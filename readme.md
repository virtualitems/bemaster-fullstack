# Prueba técnica desarrollador fullstack

# Punto 1

Para ejecutar la aplicación, se deben ejecutar los siguientes comandos:

```bash
npm install
npm start
```

A continuación, se debe abrir el navegador en la siguiente dirección:

```bash
http://localhost:3000
```
